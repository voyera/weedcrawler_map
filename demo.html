Usage Examples

Basic Setup with Language Support

<script src="https://cdn.jsdelivr.net/gh/voyera/weedcrawler_map@1.1.0/cannabis-store-map.1.1.0.js"></script>
<script>
    const map = CannabisStoreMap.createFromAPI(
        'map-container',
        'https://api.weedcrawler.ca/map/your-api-key',
        {
            language: 'fr', // 'en' or 'fr'
            theme: 'dark',
            provinces: ['QC', 'ON']
        }
    );
</script>

Dynamic Language Switching

// Switch to French
map.setLanguage('fr');

// Switch to English
map.setLanguage('en');

// Get current language
const currentLang = map.getLanguage();
